Subject Description

The Fundamentals of Computer Engineering course introduces the essential concepts that form the basis of modern computer systems. It covers both theorical foundations and practial aspects, helping students understand how computers work from the ground up.

Units Covered

  • Introduction to Computer Architecture
  • Number Systems and Data Representation
  • Boolean Algebra and Logic Gates
  • Basic Digital Circuits

Practical Assigments

  • Building logic circuits using simulation software
  • Binary and hexadecimal conversion exercises
  • Implementing a simple ALU (Arithmetic Logic Unit)
  • Designing a basic CPU in a digital logic simulator
