Subject Description

The Fundamentals of Computer Engineering course introduces the essential concepts that form the basis of modern computer systems. It covers both theorical foundations and practial aspects, helping students understand how computers work from the ground up.

Units Covered

  • Computer engineering in society
  • Current paradigms and trends in system applications
  • Information and data
  • Software development
  • Web applications
